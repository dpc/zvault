zvault-genkey(1) -- Generate a new key pair
===========================================

## SYNOPSIS

`zvault genkey [OPTIONS] [FILE]`


## DESCRIPTION

This subcommand generates a new key pair, prints it to console and optionally
writes it to the given file `FILE`.


## OPTIONS

  * `-h`, `--help`:

    Prints help information
